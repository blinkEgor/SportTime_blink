# SportTime_blink
This my own workout web-app, For personal use

## Структура проекта

- `assets/` - Медиафайлы:
  - `icons/` - PNG иконки
  - `images/` - Изображения упражнений

- `src/` - Исходный код:
  - `js/` - Основная логика
  - `styles/` - CSS-стили
  - `utils/` - Вспомогательные скрипты

- `pages/` - HTML-страницы:
  - `exercises/` - Упражнения ( dips.html, pull-ups.html, ... )

- `docs/` - Документация

## Запуск

Просто откройте `index.html` в браузере

## Особенности проекта
- Персональный трекер тренировок
- Оффлайн-работа
- Адаптивный дизайн

## Технологии
- Чистый JavaScript
- CSS без препроцессоров
- Semantic HTML
